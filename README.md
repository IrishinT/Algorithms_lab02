# Лабораторная работа №2 — Алгоритм Евклида в Windows Forms

## Описание проекта

Данный проект представляет собой графическое приложение на **C# (Windows Forms)**, реализующее алгоритм Евклида для двух целых чисел.
Приложение позволяет:

1. вычислять НОД и НОК
2. отображать все шаги алгоритма Евклида
3. визуализировать процесс нахождения НОД методом вычитания

Программа имеет интерфейс с вкладками.

---

## Функциональность

### Вкладка «Калькулятор НОД/НОК»

* ввод двух целых чисел
* вычисление НОД алгоритмом Евклида
* вычисление НОК через НОД
* проверка корректности ввода

### Вкладка «Шаги алгоритма Евклида»

* вывод всех делений с остатком в `ListBox`
* формат: `a = b * q + r`
* последняя строка — значение НОД

### Вкладка «Визуализация вычитанием»

* графическое представление алгоритма
* прямоугольники пропорциональны числам
* на каждом шаге из большего числа вычитается меньшее
* финальный прямоугольник соответствует НОД
* используется масштабирование для больших значений

---

## Используемые формулы

НОД вычисляется алгоритмом Евклида:

```
while (b != 0)
{
    int temp = b;
    b = a % b;
    a = temp;
}
```

НОК вычисляется через НОД:

```
LCM(a, b) = |a * b| / GCD(a, b)
```

---

## Структура проекта

* `CalculatorForm` — основная форма приложения
* `btnCalcGCDLCM_Click()` — обработка ввода и запуск вычислений
* `GCD(int a, int b)` — метод вычисления НОД
* `LCM(int a, int b, int gcd)` — метод вычисления НОК
* `ShowSteps(int a, int b)` — вывод шагов алгоритма в `ListBox`
* `PrepareVisualization(int a, int b)` — подготовка данных для отрисовки
* `visualPictureBox_Paint()` — графическая визуализация через `Graphics`

---

## Используемые элементы управления

* `TextBox` — ввод чисел
* `Button` — запуск вычислений
* `Label` — вывод результатов
* `ListBox` — отображение шагов алгоритма
* `PictureBox` — область для графики

## Требования к вводу

* вводятся целые ненулевые числа
* при некорректном вводе выводится сообщение об ошибке
